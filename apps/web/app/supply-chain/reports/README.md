# Amdox ERP - AI-Powered Cloud ERP Suite

Amdox ERP is a cloud-based enterprise resource planning system built using Next.js, NestJS, PostgreSQL, and Prisma. It includes Finance, HR & Payroll, and Supply Chain modules with live dashboards, authentication, reports, and CRUD operations.

## Features

- JWT Authentication
- Main ERP Dashboard
- Finance Dashboard
- HR & Payroll Dashboard
- Supply Chain Dashboard
- Inventory Management
- Products CRUD
- Vendors Management
- Purchase Orders
- Low Stock Reports
- Swagger API Documentation

## Technology Stack

### Frontend
- Next.js
- React
- TypeScript
- Tailwind CSS
- Axios
- Zustand
- Recharts
- Lucide React

### Backend
- NestJS
- TypeScript
- Prisma ORM
- PostgreSQL
- JWT Authentication
